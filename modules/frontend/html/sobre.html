O que é Uniwallet™?

Uniwallet™ é uma aplicação feita para os universitários poderem fazer compras dentro das lojas participantes de sua univiersidade, com mais tranquilidade e sem pegar filas.

O que é o nosso projeto?

O nosso projeto é uma aplicação de compras de produtos que utiliza uma moeda virtual que chamamos de Unicoins. Ok e é seguro fazer compras nele? Sim, utilizamos o sistema de Blockchain sem precisar de intermediários durante as transações, trazendo segurança total para as suas compras. Se você te dúvidas de como funciona o Blockchain, talvez você não saiba, mas poderá sanar as suas dúvidas no site www.google.com.br.

Quem somos nós?

Somos estudantes de Engenharia de Redes de Telecomunicações da UnB, a melhor engenharia(Chupa elétrica). Dedicamos muito tempo para desenvolver esta aplicação para facilitar a vida de vários estudantes e ganhar muito dinheiro( Principalmente o segundo motivo).

Como ser um colaborador?

Se você é um cara que manja de programação e gostaria de participar do Uniwallet™, mande um e-mail para "Criaraporradoe-mail@bufallobiu.com". Iremos entrar em contanto com você dentro de 96 Horas. Estaremos anciosos para você trabalhar conosco.
