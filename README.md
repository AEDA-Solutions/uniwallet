# Uniwallet
A project to get rich.

## Requirements

 * Python3
 * Mysql
 * A browser

## How to run

 * Open the terminal and go into uniwallet root directory.
 * Then type this:
 ```
python3 ./main.py
 ```
 * If nothing goes wrong the server will be available through the browser.
 * Just access the url 
 ```
 http://localhost:8000
 ```
